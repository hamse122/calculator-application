# Contributing to Apple Calculator

Thank you for your interest in contributing to the Apple Calculator project! This document provides guidelines and instructions for contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/calculator-application.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development Workflow

### Making Changes

- Write clear, readable code
- Follow the existing code style
- Add tests for new features
- Update documentation as needed

### Testing

Run tests before committing:
```bash
npm test
```

### Code Style

Format your code:
```bash
npm run format
```

Check for linting errors:
```bash
npm run lint
```

### Commit Messages

Use conventional commit format:
- `feat:` for new features
- `fix:` for bug fixes
- `docs:` for documentation
- `style:` for formatting
- `refactor:` for code refactoring
- `test:` for tests
- `chore:` for maintenance

## Submitting Changes

1. Ensure all tests pass
2. Format and lint your code
3. Commit with clear messages
4. Push to your fork
5. Open a Pull Request

## Questions?

Feel free to open an issue for questions or discussions.
