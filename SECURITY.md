# Security Policy

## Supported Versions

Currently, we are actively supporting the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it to the project maintainers via email or GitHub Security Advisories.

Please do not open a public issue for security vulnerabilities.

## Security Best Practices

- Always use the latest version of dependencies
- Review code changes before merging
- Follow secure coding practices
