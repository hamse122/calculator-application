# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Apple-style Calculator
- Basic arithmetic operations (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, ln, log, √, x², xⁿ, 1/x)
- Memory operations (M+, M-, MR, MC)
- History tracking with localStorage persistence
- Dark/light theme toggle
- Keyboard input support
- Clipboard copy/paste functionality
- Comprehensive test suite
- Storybook component stories
- Full accessibility support (ARIA labels, keyboard navigation)

### Features
- Responsive design for mobile and desktop
- Apple-inspired UI with smooth animations
- Precise decimal arithmetic using decimal.js
- Error handling for edge cases
- Auto-scaling display for large numbers
